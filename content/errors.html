Tratamento de Erros
A api foi desenvolvida pensando em oferecer a melhor forma de tratamento de erro para o desenvolvimento de api's futuras. Por isso, toda ação realizada sempre retornará o status (success/error) além de retornar o status 200(success/OK) ou 400(error).
Exemplo de Sucesso
{ "status": "success" }
Exemplo de Erro
{ "status": "error" "error": "invalid_token" }
