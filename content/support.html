Suporte
Durante a participação na Hackathon, focamos em desenvolver o sistema com integração com o mercado livre, para que pudéssemos apresentar o sistema com mais funcionalidades. Por isso, estamos trabalhando ainda no suporte de outras plataformas, além de desenvolver mais possibilidades de respostas e adicionar mais suportes a outros atributos.
Atributos Suportados Atualmente
Código do Atributo Variações de entrada
 DIMENSIONS  Tamanho / Dimensões
 COLOR  Cor / Coloração
 BRAND  Marca / Fabricante
 MODEL  Modelo
 WEIGHT  Peso / Pesado
 PRICE  Preço / Valor / Custo / Quanto custa (Adicionado pelo sistema)
 DEPTH  Profundidade / Comprimento
 HEIGHT  Altura
 WIDTH  Largura
