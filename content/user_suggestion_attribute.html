/user/suggestion/attribute
POST
TOKEN-NEEDED
Aceita/Nega uma sugestão de atributo dada pela Kate. Caso ela não tenha sugerido um valor, o usuário poderá alterar esse valor, que será enviado pelo atributo "value"
Exemplo de Entrada
{ "suggestion": 1, "value": "13mm", "accept": true }
suggestion:
Identificador da sugestão
Exemplo de Saída
{ "status": "success" }
status:
Status de retorno (success/error)
error:
Código do erro (caso exista)
